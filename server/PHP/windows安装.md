
# windows 安装PHP

下载地址：

    https://windows.php.net/
    https://windows.php.net/download/

扩展下载地址：
    
    https://windows.php.net/downloads/pecl/releases/


设置环境变量

(我的电脑)右键 -> 属性 -> 高级系统设置 -> 高级 -> 环境变量

注意：用户变量 只有当前用户可用，系统变量 所有用户都可用。

新建 kye => value 对应的环境变量即可。

如果是路径变量，则新建Path的环境变量路径即可。
